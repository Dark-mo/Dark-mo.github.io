微积分系列（一） 某些基础概念

发表于 2024-01-13

这篇文章记录了一些我认为在学习微积分之前，十分重要的基础概念，对微积分进行更加严谨的理解有十分重要的作用

实无穷与潜无穷

关于实无穷与潜无穷，文心一言是这样说的

而明显的，潜无穷在我看来更加合理。因为潜无穷是在用有穷推断无穷。而有穷的运算法则我们就非常熟悉了，至于如何推断，请看 极限值与真实值（双重归谬法）

而实无穷是确确实实的用无穷这一个我们不了解的领域。所以贸然使用就会引出许多悖论

收敛

收敛这一数学名词，百度百科的定义是这样的

咱们可以通过一个例子感受收敛的重要意义

ps.以下的视频只是我随便找的一个视频

在这道题中很明显那条三角形的斜边应该为5，但为什么它又得7呢 就是因为它没有收敛

在开始时（两个台阶时）它的长度和为7，而明显的，最终斜边长应小于7，但随之台阶数的不断增加，它的长度却没有变小（虽然更像是直线了，像是更光滑了）

所以这种方法不合理

ps.这里就用了潜无穷的思想，用有穷推断无穷

连续性

关于函数的连续性，文心一言是这样说的

开始时我们只需要了解并承认一些初等函数（整体或某一部分上）具有连续性即可, 而连续性也为最终的双重归谬法提供了坚实的基础

极限值与真实值（双重归谬法）

双重归谬法，用福尔摩斯的名言便是“当你排除一切不可能的情况，剩下的，不管多难以置信，那都是事实。”

举个例子

定义 f(x) = 1/x

①f(x) < 0, f(x)∈R 等价于 x < 0, x∈R

②f(x) > 0, f(x)∈R 等价于 x > 0, x∈R

ps.①②意味着任意的 f(x)为非0实数，x一定也为非0实数

③在f(x) x>0 时，x越大，对应的 f(x)越小

这也让我们看出了极限值与真实值之间的关系

即在连续的函数中，x趋近于n时函数的极限值就是x=n时函数的真实值（注意这里是“就是”而非“近似”）
