海伦-秦九韶公式的证明（初中版）

发表于 2023-06-28 更新于 2023-06-29

海伦-秦九韶公式是一个非常有趣的公式，可以通过三角形的三条边求出三角形的面积，在我学的初中课本中虽然有提到但没有证明方法。现在我就写一下我初中写的证明方法

构造图形

首先，我们需要画一个三角形ABC，设A点的对边是a，B点的对边是b，C点的对边是c

证明余弦定理

过A作AH⊥BC于H,设BH为d

因为 $BC=a,BH=d$

所以 $CH=BC-BH=a-d$

因为 $AB^2=BH^2+AH^2$

所以 $AH^2=AB^2-BH^2=c^2-d^2$,

因为 $CH^2=(a-d)^2=a^2-2ad+d^2$

所以 $b^2=AC^2=AH^2+CH^2=c^2-d^2+a^2-2ad+d^2=a^2+c^2-2ad$

因为 AH⊥BC

所以 ∠AHB = 90°

所以 cosB = $\frac{BH}{AB}$

所以 d = BH = AB cosB = c cosB

所以 $b^2=a^2+c^2-2ad=a^2+c^2-2ac*cosB$

所以 $b^2=a^2+c^2-2ac*cosB$

证明海伦-秦九韶公式

因为余弦定理可知在三角形ABC中已知abc可求cosB

即 cosB = $\frac{a^2+c^2-b^2}{2ac}$

又因为 $sin^2B+cos^2B=1$

所以 $sin^2B=1-\frac{(a^2+c^2-b^2)^2}{4a^2c^2}$

所以 $AH^2 = c^2sin^2B=c^2-\frac{(a^2+c^2-b^2)^2}{4a^2}=\frac{4a^2c^2-(a^2+c^2-b^2)^2}{4a^2}=\frac{(2ac)^2-(a^2+c^2-b^2)^2}{4a^2}=\frac{(2ac+a^2+c^2-b^2)(2ac-a^2-c^2+b^2)}{4a^2}=\frac{[(a+c)^2-b^2][b^2-(a-c)^2]}{4a^2}=\frac{(a+b+c)(a-b+c)(a+b-c)(-a+b+c)}{4a^2}$

所以 $AH=\sqrt{\frac{(a+b+c)(a-b+c)(a+b-c)(-a+b+c)}{4a^2}}$

所以 $三角形ABC的面积=\frac{1}{2}aAH=\sqrt{\frac{a^2}{4}}\sqrt{\frac{(a+b+c)(a-b+c)(a+b-c)(-a+b+c)}{4a^2}}=\sqrt{\frac{(a+b+c)(a-b+c)(a+b-c)(-a+b+c)}{16}}=\sqrt{\frac{a+b+c}{2}\frac{a-b+c}{2}\frac{a+b-c}{2}\frac{-a+b+c}{2}}=\sqrt{\frac{a+b+c}{2}\frac{(a+b+c)-2b}{2}\frac{(a+b+c)-2c}{2}\frac{(a+b+c)-2a}{2}}=\sqrt{\frac{a+b+c}{2}(\frac{a+b+c}{2}-a)(\frac{a+b+c}{2}-b)(\frac{a+b+c}{2}-c)}$

设 $p=\frac{a+b+c}{2}$

所以 $三角形ABC的面积=\sqrt{p(p-a)(p-b)(p-c)}$
